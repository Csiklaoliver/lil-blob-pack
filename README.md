## What is Lil' Blob Pack?

A growing collection of free, high-quality 3D mascots — one for each app you use every day.
Every blob ships as **FBX + GLB**, ready for Blender, Unity, Unreal, or your README.

No credit card required. I made it easy and open-soruce for everyone to use.

## 📥 How to Use

1. Go to **[blobpack.oliverprojects.tech](https://blobpack.oliverprojects.tech)**
2. Pick your blob
3. Download **FBX** (Blender, Unity, Unreal) or **GLB** (web, three.js, model viewers)
4. Drop it into your project

That's it. No sign-up, no credit card, no nonsense.

---

## 🎨 File Formats

| Format | Best for |
|--------|----------|
| `.fbx` | Blender, Unity, Unreal Engine, Cinema 4D |
| `.glb` | Web (three.js, model-viewer), Figma, Spline |
